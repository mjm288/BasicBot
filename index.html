About BasicBot

BasicBot is an online robot who is an advanced intellectual, able to compute and calculate far beyond humans. BasicBot was created to be able to aid humans in their daily lives, both as a friend and as a calculator.

How To Use BasicBot

For best results...

  • Use spaces when calculating e.g "5 + 5"
  • Don't use punctuation

Some key words:

  • time - asking for the date and time
  • 8ball - triggers 8ball mode (for randomised answers to any question)
  • + - addition operator
  • - - subtraction operator
  • * - multiplication operator
  • / - division operator
  • ** - exponential operator
  • pi - outputs pi to 15 digits
  • search - google searches in a new tab
  • reverse - reverses a string
  • generate - generates a randomised string of characters
